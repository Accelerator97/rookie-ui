# Progress 进度条
<br/>

## 条形进度条

```jsx
function ProgressExample() {
    const [count, setCount] = useState(30);
    const handleClick1 = ()=>{
        setCount(precount=>{ return precount + 10})
    }
    const handleClick2 = ()=>{
        setCount((precount)=>{ return precount - 10})
    }
    return (
        <div>
            <Button onClick={handleClick1}>+10</Button>
            <Button onClick={handleClick2}>-10</Button>
            <Progress percent={count} ></Progress>
        </div>
    );
}
```
